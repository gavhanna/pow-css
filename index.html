Regular Paragraph Text

It can be useful to have a system in place that's easy to go full-width, or have a "pow-breakout" section for calls to action or blockquotes, or other things you want to call attention to.

Any content can be set to "breakout" of the regular max width of the page content, but not reach the edges of the screen. This is useful for calls to action, or blockquotes, or other things you want to call attention to.

Any HTML is allowed in the breakout sections, as we'll see below.

Color Palette

The color palette is defined using HSL (Hue, Saturation, Lightness) values. This makes it easier to manage color contrast and create color variations. The color palette is defined using a 300, 400, and 500 value for each color. The 300 value is the lightest, the 500 value is the default, and the 400 value is the darkest.

Colours can be added or removed as needed, e.g. 600, 700 (darker) or 200, 100 (lighter).
